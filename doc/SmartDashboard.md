# Description
This is a file to indicate what should be found on the smartdashboard in competition. Everything else should be cleared, as well as prints.

# Tabs

## Technician

### Essential

- Autonomous mode (chooser)
- Limelight active (boolean)
- Is gyro calibrated (boolean)
- Manual/auto controller mapping (chooser)
- Sensor ping (boolean)
- Deep/shallow climb (chooser)

### Optional

## Driver

### Essential

- AprilTag Camera (Front)
- Is in bounds for processor (boolean)
- Driver cam
- Field (big)
- Is at desired position (auto align, boolean)
- Algae intaked (boolean)
- Coral intaked (boolean)

### Optional

- Elevator desired height (string)
- Lock ID (number)
